A modern and stylish e-commerce website dedicated to men's fashion. Discover the latest trends in clothing, shoes, and accessories with curated collections, top sales, new arrivals, and hot deals.

## Features

- **Home Page**: Eye-catching hero section with summer collection highlights
- **Collections**: Showcase of clothing, shoes, and accessories
- **Sellers Section**: Top Sales, New Arrivals, and Hot Sales with product ratings and prices
- **News**: Latest fashion trends and articles
- **Contact**: Interactive map and contact form
- **Responsive Design**: Optimized for all devices
